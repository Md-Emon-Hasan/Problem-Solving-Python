# HackerRank – Python Division

## Problem

Given two integers `a` and `b`, print:

1. Integer division `a // b`
2. Float division `a / b`

---

# Theory

Python has two types of division:

| Operator | Meaning |
|---|---|
| `//` | Integer division |
| `/` | Float division |

Example:

```
4 // 3 = 1
4 / 3 = 1.3333
```

---

# Approach

1. Read integer `a`
2. Read integer `b`
3. Print integer division
4. Print float division

---

# Python Implementation

```python
a = int(input())
b = int(input())

print(a // b)
print(a / b)
```

---

# Key Idea

Division operators:

```

/
```
