# HackerRank – Say Hello, World! With Python

## Problem

Print the following message:

```
Hello, World!
```

No input is required.

---

# Theory

Python uses the `print()` function to display output.

Syntax:

```
print("text")
```

---

# Python Implementation

```python
print("Hello, World!")
```

---

# Output

```
Hello, World!
```

---

# Key Idea

Basic output pattern:

```
print("message")
```
